# Bird Annotation Recovery

Automated tool to recover bounding box annotations from dot-annotated aerial wildlife images.

## Problem

Historical wildlife surveys contain dot annotations instead of proper bounding boxes, making them unusable for modern ML training.

## Solution

Automated pipeline that converts dot annotations to ML-ready bounding boxes.

## Tech Stack

| Component | Technology |
|-----------|------------|
| Segmentation | SAM (Segment Anything) |
| Image Processing | OpenCV, NumPy |
| ML Framework | PyTorch |
| Object Detection | DeepForest |
| Data Handling | Pandas |
| Visualization | Matplotlib |
